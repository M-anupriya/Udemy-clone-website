# Udemy-clone-website
Udemy Clone Website

This project is a simple Udemy-inspired web page developed using HTML and CSS. It consists of multiple sections, including a navigation bar, Recommended courses, and Popular courses. The website is designed with a clean and structured layout using Flexbox to ensure proper alignment and spacing. It also includes responsive design techniques to provide a smooth and user-friendly experience across different devices. This project demonstrates foundational front-end development skills, including webpage structure, styling, layout management, and responsive design.

How to View

You can view this project by cloning the repository and opening the index.html file in any web browser. Alternatively, the website can be accessed through GitHub Pages if enabled.
